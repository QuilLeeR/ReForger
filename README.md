# ReForger
ReForger is a WinUI 3 / .NET 8 desktop tool for creating, editing, installing, and reverting Assassin's Creed Shadows mods. It works with the game's Anvil Pipeline v42 .forge containers and uses ReForger's own compact RFGB (.rfgb) mod package format, with support for tweak lists, Choice Sets, embedded files, backups, and safe apply/revert workflows
